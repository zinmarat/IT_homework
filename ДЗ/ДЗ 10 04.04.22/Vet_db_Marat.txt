4) создаем таблицу питомцев pet со столбцами id, name, client, doc, cure. Последние три столбца это внешние ключи на другие таблицы в БД


CREATE TABLE pet ( 

			id SERIAL PRIMARY KEY, name VARCHAR(30), 
                  client INTEGER, FOREIGN KEY (client) REFERENCES client (Id),
                  doc INTEGER, FOREIGN KEY (doc) REFERENCES doc (Id),
                  cure INTEGER, FOREIGN KEY (cure) REFERENCES cure (Id) 

			)

Заполним таблицу домашних питомцев где укажем их имя (name) и ссылку на хозяина (client), ветерианара (doc) и оказанное лечение или услугу (cure)
